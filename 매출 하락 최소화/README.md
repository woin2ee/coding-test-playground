## Review
제출 전 주어진 테스트 케이스는 통과했지만, 제출 후 3 ~ 20번 테스트를 시간 초과로 실패했다.  
아마 무작정 재귀를 통해 푸는 것이 아니라 케이스를 나누어 시간 최적화를 해야 할 것 같다.  
그래도 N중 for문이나 재귀의 개념에 대해 좀 더 생각해보는 계기가 되었다.
